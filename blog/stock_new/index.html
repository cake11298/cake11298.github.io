討論外資未平倉CALL/PUT口數

最近在期貨選擇權的籌碼裡面學到，可以大概看外資未平倉的CALL/PUT口數大概觀察外資想保護的部位，而且可以大概看出外資預期最近台股走勢的一小部分。

使用 FinLab API 觀察大盤情勢串接券商軟體

使用XGBoost訓練的AI，觀察每一支股票在基本面上的成長率，並使用Fred儀錶板的美國短期長期債券利率是否倒掛的方法，觀察目前整體經濟與基本面。

賣出大型股 買入小型股 策略(類價值投資)

高風險 高報酬 意指 高平均預期報酬

所以 買高風險的東西 在好時機時 你可拿到高報酬的一面 風險折線率

但 今天如果是股災 或 壞時機 那你的報酬會雪上加霜

小型股策略(買進小型股 賣出大型股)

控制了市場所有因素，小型股票的獲利能力仍然顯著優於大型股票

價值效果

公司資產對市值的比率 如果接近1 那就是高報酬股票

市場價格低估 可能來源於高風險 所以你吃下這個高風險 預期獲得高報酬的獲利

這是一樣高風險投資 所以應該預期這個投資在時機好的時候才會大賺

否則 應該會面臨 帳面上損失很多的問題

動能效果

選一些不錯的公司

把每間公司的 前3~12月 平均股價報酬率算出來

買進 高平均報酬率的公司

賣出 低平均報酬率的公司

市場上 有限的注意力 即 保守性 資訊反應在股價上的時間會有時差

有些人認為可能漲過了 有些人還在想 這樣算是完全反應完了嘛? 所以這樣追漲的操作 有這個結果不太過分
